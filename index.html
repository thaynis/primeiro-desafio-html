Estrutura de títulos e subtítulos

Criar uma página na web pode parecer uma tarefa difícil, mas utilizando o HTML conseguimos estruturar de maneira simples com o uso de tags, em primeiro momento as tags podem parecer difíceis de encontrar, mas temos muitas ferramentas de ajuda, como o site W3schools, ou até mesmo mais atualmente o ChatGPT. Ao criar a página, é fundamental organizar o conteúdo de forma clara e hierárquica. No HTML, essa organização é feita por meio das tags de títulos e subtítulos, que vão de <h1> até <h6>. O < h1> é o título principal da página, ele representa a ideia central ou o tema mais importante do conteúdo. A partir daí, os níveis vão descendo com <h2> até <h6>, que funcionam como subtítulos ou seções menores.

Estrutura de parágrafos

A estrutura de parágrafos é mais simples, usando a tag <p> é possível determinar onde começa e termina o parágrafo proposto dentro do corpo do texto no HTML.

Inserindo links

Para inserir links no seu texto/corpo do HTML, é necessário usar a tag <a>, o significa em inglês é anchor,pois ela "ancora" o link em um lugar específico. Para funcionar, usamos o atributo href, que indica o destino do link.
